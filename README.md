# LearnAid

## Description
LearnAid helps you learn all the topics according to your ideas with a card system.


## Installation

### Start projekt
* git clone https://github.com/Wummer94/LearnAid
* cd LearnAid
* npm i
* ionic serve
* Go to: http://localhost:8100

### Start server 
* cd LearnAid
* cd server
* npm i
* node index.js
